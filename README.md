# RS-WZ3-N01-1-Temperature-vibration_sensor_reading_registers_and_trigger_warning


Python-based vibration monitoring system using Modbus RTU communication. The script reads X, Y, Z vibration speed and displacement from an industrial vibration sensor via RS-485, evaluates machine condition using ISO vibration thresholds (Normal / Warning / Danger), and logs the data with timestamps to a CSV file for analysis.
